ICC Cricket World Cup 2023 Analysis

Project Overview

This project analyzes the ICC Cricket World Cup 2023 dataset using Python to uncover insights into team performance, player statistics, match outcomes, batting and bowling trends, and venue-wise performance. The analysis helps identify key factors that influenced match results and overall tournament success.

Objectives

* Analyze team and player performances throughout the tournament.
* Identify top run scorers and wicket-takers.
* Evaluate batting and bowling trends.
* Examine venue-wise match statistics.
* Compare team win percentages and consistency.
* Generate actionable insights through data visualization.

## Dataset Information

The dataset contains:

* Match Details
* Team Information
* Player Statistics
* Batting Performance
* Bowling Performance
* Venue Details
* Match Results

## Project Workflow

### 1. Data Collection

* Imported ICC World Cup 2023 dataset.
* Loaded data into Pandas DataFrames.

### 2. Data Cleaning

* Checked for missing values.
* Removed duplicate records.
* Corrected data types.
* Standardized column names.

### 3. Exploratory Data Analysis (EDA)

### 4. Team Performance Analysis

* Wins and losses comparison.
* Win percentage calculation.
* Team ranking based on performance.

### 5. Batting Analysis

* Top run scorers.

* Strike rate comparison.
* Boundary analysis (4s and 6s).

### 6. Bowling Analysis

* Top wicket-takers.
* Economy rate comparison.
* Best bowling figures.

### 7. Venue Analysis

* Average first innings score.
* Venue impact on match outcomes.

### 8. Match Analysis
